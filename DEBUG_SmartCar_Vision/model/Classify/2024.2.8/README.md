2024.2.8

未继续训练目标板数据，经过实测无白边目标板 有11/12的正确率(在有自然光干扰的条件下) 输入图像像素320 * 240， 模型输入层96 * 96

经过小类和大类混合训练，有字母识别为数字的问题
为了避免数字三大类和字母小类混合识别, 分为两个模型训练.

经过训练 图像像素为320 * 240, 输入层96 * 96识别数字预测率均在0.996以上，较稳定

同样经过训练, 图像像素为320 * 240， 输入层为96 * 96 识别字母准确率得以提高, 但是有存在M, N, H, F主要这四个字母识别容易出错.

初步预估原因:
    由于训练图像像素为320 * 240而输入层为96 * 96 使得这几类的图像特征区分不明显，导致图像识别正确率降低

解决方案:
    将训练图像设置为96 * 96 与输入层对应
